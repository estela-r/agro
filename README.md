
# AGRO 
Sample PHP REST API application built using Zend Expressive and Doctrine ORM.

## Setup
```
git clone git@github.com:estela-r/agro.git
cd agro
composer install
```
## Usage
The API exposes the following endpoins:

 POST /api/plots

 GET /api/plots/{id}

 POST /api/tractors
 
 POST /api/tillage

 GET /api/tillage

## Unit tests
```
vendor/bin/phpunit test
```
